1. undefined মানে হচ্ছে একটি ভ্যারিয়েবল ডিক্লেয়ার করা হয়েছে কিন্তু কোনো মান দেওয়া হয়নি, null মানে হচ্ছে ইচ্ছাকৃতভাবে কোনো মান নেই বুঝয়

2. map() একটি অ্যারের প্রতিটি উপাদানের উপর কাজ করে এবং একটি নতুন অ্যারে রিটার্ন করে, forEach() অ্যারের প্রতিটি উপাদানের উপর কাজ করে, কিন্তু কোনো নতুন অ্যারে রিটার্ন করে না

3. == শুধু মান তুলনা করে, === মান এবং টাইপ দুটোই তুলনা করে

4. async/await ব্যবহার করলে অ্যাসিনক্রোনাস কোড সহজভাবে লেখা যায় এবং কোড পড়তে সহজ হয়

5. একটি ভ্যারিয়েবল কোথায় ব্যবহার করা যাবে তা হলো Scope
   Global Scope: যে ভ্যারিয়েবল পুরো প্রোগ্রাম জুড়ে ব্যবহার করা যায়
   Function Scope: যে ভ্যারিয়েবল শুধু ফাংশনের ভিতরে ব্যবহার করা যায়
   Block Scope:শুধু নির্দিষ্ট ব্লকের ভিতরে ব্যবহার করা যায়
